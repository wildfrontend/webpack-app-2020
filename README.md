- react env basic
  
```json
{
    "presets": [
      "@babel/preset-env", "@babel/preset-react"
    ]
}
```

### package.json

- script 
  - `yarn prettier` : format all js files in Project
  - `yarn start` : start dev server
  - `yarn build` : bundle static file
